Avantages CYIM applicables

  • Prise en charge transport en commun : 50 % du coût de l'abonnement souscrit par l'alternant (obligation légale — Code du travail, art. L3261-2).
  • Indemnité kilométrique vélo (IK vélo) : 0,25 €/km pour les déplacements domicile-travail à vélo, dans la limite d'un plafond annuel de 400 €.
  • Titres-restaurant : valeur faciale 7,50 €, dont 60 % pris en charge par l'employeur (soit 4,50 €/TR), pour les jours travaillés en présentiel.

Particularité des jours de révision

Pour le contrat d'apprentissage, l'apprenti bénéficie d'un congé supplémentaire de 5 jours ouvrables pour réviser ses examens, à prendre dans le mois précédant les épreuves (article L6222-35 Code du travail). Ces jours sont rémunérés intégralement par l'employeur. Pour la professionnalisation, aucun congé révision spécifique n'est prévu par le Code du travail (sauf disposition conventionnelle).

Documentation — Mode d'emploi

À quoi sert cet outil ?

Le simulateur permet de calculer des augmentations salariales sous plusieurs angles (du nouveau salaire au %, du % au nouveau salaire, d'un montant en € au nouveau salaire), en vérifiant automatiquement la conformité avec :

  • les minima conventionnels SYNTEC (IDCC 1486) en vigueur
  • les seuils internes CYIM (+10 % ETAM / +20 % Cadres) — marge de sûreté
  • le SMIC en vigueur (plancher légal absolu)

Les deux modes d'affichage

  • Mode Détaillé : tous les scénarios avec leurs détails (résultats mensuels/annuels, tableau de sensibilité, conversion annuel/mensuel/horaire). Idéal pour préparer un dossier d'augmentation argumenté.
  • Mode Compact : les 3 simulations principales (Salaire→%, %→Salaire, Montant→Salaire) condensées avec les verdicts SYNTEC et CYIM en pastilles colorées. Idéal pour un cycle de simulations rapides.

Logique des seuils

Pour les ETAM :

  • Plancher légal mensuel = MAX(minimum conventionnel SYNTEC 35h ; SMIC mensuel)
  • Plancher 39h = plancher 35h × (169 / 151,67)
  • Minima interne CYIM = plancher 39h × (1 + marge ETAM, par défaut +10 %)

Pour les Cadres :

  • Plancher légal mensuel = MAX(minimum conventionnel SYNTEC ; SMIC mensuel)
  • Forfait jours mensuel = plancher 35h × (1 + marge Cadres, par défaut +20 %)
  • Minima interne CYIM = forfait jours × (1 + marge ETAM, par défaut +10 %)

Quand mettre à jour les valeurs ?

  • SMIC : à chaque revalorisation officielle.
  • Grille SYNTEC : à chaque publication d'un nouvel avenant par la branche (négociation annuelle CPPNI au plus tard le 30 juin).
  • Bases horaires : rarement (uniquement si la durée légale ou la convention horaire CYIM change).
  • Marges internes : selon la politique RH interne.

Comment modifier la grille SYNTEC ?

  1. Va dans l'onglet Grilles SYNTEC
  2. Clique dans la cellule Min. conv. 35h de la ligne à modifier
  3. Saisis la nouvelle valeur. Toutes les colonnes dérivées se recalculent immédiatement
  4. La sauvegarde est automatique (stockage navigateur)

Persistance des données

Toutes tes modifications (paramètres, grilles SYNTEC, salariés saisis dans le tableau d'enveloppe) sont automatiquement sauvegardées dans le stockage local de ton navigateur. Pour partager une configuration : Paramètres → Exporter (JSON), et le collègue fait Importer (JSON).

Glossaire

  • SYNTEC : Convention collective nationale des bureaux d'études techniques, cabinets d'ingénieurs-conseils et sociétés de conseils (IDCC 1486).
  • ETAM : Employés, Techniciens et Agents de Maîtrise.
  • Forfait jours : dispositif de décompte du temps de travail en jours (218 jours/an) applicable aux Cadres autonomes (positions 3.x SYNTEC en règle générale).
  • Base 39h CYIM : les ETAM de CYIM sont contractuellement à 39h/semaine, soit 169 h/mois (vs 151,67 h pour 35 h). Les heures supplémentaires majorées (25 %) sont gérées séparément en paie.
